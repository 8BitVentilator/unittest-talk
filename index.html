Was ist ein Unit Test?

Deutsche Übersetzungen:

  • Modultest
  • Komponententest

Was ist ein Modul bzw. eine Komponente?

  • Ein Modul/Eine Komponente ist eine geschlossene Einheit von einer Funktionalität
  • Es kann z.B. eine Klasse sein...
  • ...oder sogar eine einzelne Methode bzw. Funktion
							
								public class Car
								{
									// weiterer code
			
									public void Drive()
									{
										Console.WriteLine("Wrumm!");
									}
								}

Ein Unit Test ist ein Stück Code,
der unseren Code testet

Wozu brauchen wir Unit Tests?

  • Ohne Unit Tests muss das Programm manuell getestet werden
  • Manuelle Tests sind schlecht skalierbar
  • Unit Tests geben uns schnell Feedback

Wie schreibt man Unit Tests?

Für jede Sprache gibt es unterschiedliche Frameworks. Beispeie:

C#
  • xUnit
  • NUnit
  • MS Test
Java
  • Java
  • JUnit
Python
  • unittest
PHP
  • PHPUnit

Es soll die Komponente "FizzBuzz" geschrieben, die die Funktion "Translate" (= Übersetze) besitzt. Als Parameter bekommt die Funktion eine Zahl und sie soll folgendermaßen übersetzt werden:

  • Für Vielfache von 3 gibt „Fizz“ zurück
  • Für Vielfache von 5 gib „Buzz“ zurück
  • Für Vielfache von 3 und 5 gib „FizzBuzz“ zurück
